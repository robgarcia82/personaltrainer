# Personal Trainer — Guia de Exercícios PWA

App de treino personalizado para home gym.

## Setup
- MC-1000 (Polia ajustável + Barra fixa + Paralelas)
- Halteres 2,5-24 kg
- Banco regulável
- Sandbag 10/20 kg

## Programa
- 12 semanas, periodização em blocos
- 4x/semana: Seg (Upper A), Ter (Lower A), Qui (Upper B), Sáb (Lower B)
- 32 exercícios com instruções detalhadas

## Funcionalidades
- Busca e filtros por sessão e grupo muscular
- Instruções passo a passo de cada exercício
- Periodização por bloco (séries, reps, PSE)
- Timer de descanso integrado
- Funciona offline (PWA)

## Instalar no iPhone
1. Abra o link no **Safari**
2. Toque no botão de compartilhar (↑)
3. Toque em **"Adicionar à Tela de Início"**
